### PLYST
**Playlist + List = PLYST**

음악을 함께 듣고, 공유하고, 소통하는 음악 커뮤니티 플랫폼

<br/>

**Key Features**
- **플레이리스트** - Spotify 연동, 생성/공유/검색
- **AI 추천** - ChatGPT 기반 맞춤 음악 추천
- **스테이션** - 실시간 동시 감상 & 채팅
- **소셜** - 좋아요, 댓글, 팔로우
- **자막** - Whisper AI 실시간 번역

</td>
<td width="50%">

```

**Infra**
```
